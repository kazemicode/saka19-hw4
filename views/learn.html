NLTK

The Natural Language Toolkit (NLTK) is a popular opensource platform for building Python programs for NLP. Libraries exist for both Python 2 and Python 3 with extensive libraries for word classification, sentence parsing, and more.

Stanford NLP Group

The Stanford NLP Group also provide opensource access to their CoreNLP API for developing NLP applications in Java; the API has similar capabilities to NLTK, and also includes a named entitiy recognizer.
